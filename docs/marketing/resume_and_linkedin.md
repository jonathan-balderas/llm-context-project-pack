## Resume (best placement)

- "Professional Development"
or
- "Projects" (if you already have that section)

- Built a **Project Pack workflow** for using ChatGPT on recurring professional tasks, using an ownership-based context index, thread lifecycle rules, and deterministic patch outputs (tracker/reminders/schedule).
- Created a lightweight toolchain (Python scripts) to **validate context integrity** and **version/update headers**, enabling weekly refreshes with low maintenance overhead.

Optional (if true for you - keep generic, avoid sensitive details):
- Designed a "single-row" pipeline tracker + reminder export flow to reduce admin overhead and keep workstreams moving.
